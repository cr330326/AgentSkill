# 故障排查问题模板

## 核心问题（必问）

1. **症状是什么？** 具体的错误信息、异常行为、截图。
2. **什么时候开始的？** 一直存在还是最近才出现？最近有什么变更？
3. **能复现吗？** 每次都能复现 / 偶发 / 只在特定条件下？复现步骤是什么？
4. **影响范围多大？** 所有用户 / 部分用户 / 特定环境？
5. **已经尝试了什么？** 哪些排查手段已经用过？结果是什么？

## 补充问题（按需追问）

6. 相关的日志和监控数据有吗？能贴出来吗？
7. 最近有没有代码变更、配置变更、基础设施变更？
8. 其他相关服务/组件是否正常？
9. 有没有类似问题的历史记录？之前是怎么解决的？
10. 当前有没有临时的 workaround 在用？

## 跳过条件

- 如果用户已经贴了完整的错误日志和复现步骤 → 跳过问题 1-3，直接开始分析
- 如果是开发环境的问题 → 简化问题 4（通常只影响开发者本人）
- 如果用户说"刚部署就挂了" → 聚焦问题 7（最近变更）
