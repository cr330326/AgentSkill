# 配置初始化问题模板

## 核心问题（必问）

1. **目标环境是什么？** 开发 / 测试 / 生产？操作系统？
2. **已有哪些基础设施？** 数据库、消息队列、缓存、CDN 等。
3. **需要哪些环境变量或密钥？** 列举需要配置的第三方服务。
4. **兼容性要求？** 需要兼容哪些版本的语言/运行时/浏览器？
5. **安全要求？** HTTPS、CORS 策略、认证方式、密钥管理方式。

## 补充问题（按需追问）

6. 日志和监控方案是什么？（如 ELK、Prometheus、Sentry）
7. CI/CD 用什么工具？（GitHub Actions / GitLab CI / Jenkins）
8. 需要区分多环境配置吗？（dev/staging/prod）
9. 有没有现有的配置模板或 boilerplate 可以参考？
10. Docker 化还是裸机部署？

## 跳过条件

- 如果是本地开发环境初始化 → 简化问题 5，跳过问题 6-7
- 如果用户指定了工具（如 "用 Vite 初始化"）→ 聚焦问题 1、3
