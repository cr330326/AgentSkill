流水线包含你提到的 6 个阶段：签署合同 → 创建客户账号 → 配置权限 → 导入历史数据 → 培训 → 正式上线。

其中 3 个阶段标记了 `[CHECKPOINT]`：

1. **签署合同** — 这是整条流水线的前提，合同条款和服务范围需要人工确认后才能启动后续工作。
2. **导入历史数据** — 涉及外部数据源和生产环境写入，必须先在测试环境验证、由客户抽检确认后再正式导入。
3. **正式上线** — 不可逆操作，上线前需要确认所有前置阶段完成、回滚方案就绪。

每个阶段都定义了目标、操作步骤和输出产物，CHECKPOINT 阶段额外包含检查要点。你可以根据实际业务调整具体操作细节和检查项。
